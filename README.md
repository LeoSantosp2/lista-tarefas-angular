### Storage

## Get Started
1. Clone this repository
```bash
git clone https://github.com/LeoSantosp2/lista-tarefas-angular.git
```

2. Download the packages
```bash
npm i
```

Or

```bash
npm install
```

3. Execute the bellow command
```bash
npm start
```

Or

```bash
ng serve --open
```

4. Access the link `http://localhost:4200`
